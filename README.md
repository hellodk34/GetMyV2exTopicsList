# GetMyV2exTopicsList
get my v2ex topics list. 快速获取 v2ex 上我收藏的帖子列表

---

由于 v2ex 的收藏页面 https://www.v2ex.com/my/topics 没有检索功能

随着收藏的帖子越来越多，找到曾经收藏的帖子越来越困难。于是做了这个小工具。
